# Credit Risk Analysis

This repository contains a small credit risk analysis project using the German credit dataset.

Files of interest:
- `credit_risk_analysis.py`: main analysis script.
- `data/german_credit_data.csv`: dataset used by the analysis.
- `credit_risk-analysis/`: additional project files.

Quick start
1. Create and activate a Python virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Install dependencies (if the project has a `requirements.txt`):

```bash
pip install -r requirements.txt
```

3. Run the analysis:

```bash
python credit_risk_analysis.py
```

Notes
- The dataset is included in `data/german_credit_data.csv`.
- Add a `requirements.txt` if you want reproducible installs.
